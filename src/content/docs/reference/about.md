本サイト「[Begleri Docs](https://begleridocs.github.io/)」についての情報。

#### サイト名
Begleri Docs

#### 内容
現在のベグレリに関する情報や、ベグレリ大会・技術に関する情報ををまとめたサイト。ベグレリの2大大会であるSlingSlam・SingleGripOpenの歴代入賞者の入賞動画を確認することができるほか、ベグレリを楽しむうえで有益と思われる情報をまとめている。

#### 姉妹サイト
本サイトの姉妹サイトとして、[BegleriInfo](https://sites.google.com/view/begleri-info/home)がある。BegleriInfoはベグレリを始める人向けに役立つサイトを志向し、情報を厳選して掲載しているが、本サイトは情報を記録することに焦点を当て、できるだけ多くの情報を掲載している。

#### 管理者
本サイトは、[なおなり](https://naonari.netlify.app/)が管理しています。本サイトに関するお問い合わせは、[Instagram](https://www.instagram.com/1021z_ani/)または[ウェブフォーム](https://tally.so/r/Np5P0O)からお願いします。ベグレリに関する質問や相談などもできる限りお答えします。

#### 本サイトについて
本サイトは、高速な表示速度が特徴の[Astro](https://astro.build/)というフレームワークを使用しており、快適に閲覧できます。[Starlight](https://starlight.astro.build/)というドキュメントテーマを採用しており、サイドバーや検索機能を使用して便利に情報収集ができます。

また、本サイトはGitHubを使用して管理しているため、共同編集が可能です。本サイトの編集を希望する方は管理者までご連絡ください。なお、共同編集にはGitおよびGitHubを使用する必要があります。
