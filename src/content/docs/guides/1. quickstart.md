## ベグレリの入手
ベグレリは、専門メーカーが販売している商品を買うことで入手できる。また、材料を揃えれば自作をすることも可能である。

ベグレリは、ビーズの重さや紐の長さなどに規定はなく自由であるが、一般に扱いやすいとされるベグレリの特徴を列挙すると以下のようになる。
- ビーズは片側8～12グラム程度
- 紐の長さは15cm程度
- 紐の太さは4mm程度

### 自作（モンキーフィスト）
以下の材料を揃えれば、ベグレリを自作することが可能である。
- パラコード（太さ4mm程度）
- 重りになる玉（ビー玉など）
- ライター（紐端の処理に必要）

ビー玉などの重りとなる玉を中心に「モンキーフィストノット」という編み方でパラコードを編み、紐の長さを揃えることでベグレリを作ることができる。

### 自作（金属製）
以下の材料を用いて自作できる
- 金属スペーサーや六角ナット
- パラコード（太さ4mm程度）
- ライター（紐端の処理に必要）

### ベグレリの購入
ベグレリの大手メーカーは、[aroundsquare](https://www.aroundsquare.com/)である。aroundsquareはカナダの会社で、ベグレリをはじめとした様々なスキルトイを販売している。
日本では[SummerSalts](https://summersalts.stores.jp/)、[SPINGEAR](https://spingear.jp/)が代理店として取り扱っている。
代理店に欲しい商品がない場合は、aroundsquare公式HPから直接購入することになる。代理店を介さずaroundsquare公式HPから購入した場合は、到着までに約2～3週間程度、送料は1500円ほどかかったことがある。
以下のベグレリは特にクセがなく扱いやすい
- Mangas
- Mini Titans
- Mini Everyman
- Herc Core
- Hype mini

## 準備
ベグレリを入手したら、紐の長さを調整する必要がある。

一般に「標準的な紐の長さ」の計測方法として、ベグレリの紐の露出している部分の長さを、手を軽く広げた際の人差し指の先と小指の先までの長さに合わせる方法が知られている（ロングゲーム）。

（その他の方法はRedditの[Begleri String Length Guide](https://www.reddit.com/r/begleri/wiki/stringlength/)を参照）

なお、長さ調整をする際に切った紐端をそのままにしておくと、紐がほつれてしまうため、ライターの火で軽く溶かして処理するとよい。

## 用語・概念
ベグレリを練習するうえで知っておくと便利な用語・概念を解説する。
### グリップ
ベグレリの持ち方・持つ位置をグリップと呼ぶ。グリップには以下の種類がある。
| 使用する指 | グリップ名 | 英数字による表記 |
| ---- | ---- | ---- |
| 親指と人差指 | トップグリップ | T1 |
| 人差し指と中指 | ハイグリップ | 12 |
| 中指と薬指 | ミッドグリップ | 23 |
| 薬指と小指 | ローグリップ | 34 |
| 小指と手のひら | ボトムグリップ | P4 |

:::note
※英数字による表記とは、各指に英数字を割り当て、それによりグリップを表記する方法である。親指をT、人差し指を1、中指を2、薬指を3、小指を4、手のひらをPとし、それらの組み合わせでグリップを表記している。
:::

:::caution
（[musclebonesチュートリアル](https://www.youtube.com/watch?v=-2DVEFePGgc&list=PLA3SEfotogzcBDTDSMjKXQIRtl37BiTuA&index=2)ではトップグリップを「ハイグリップ」、ハイグリップを「スタンダードグリップ」と呼んでいるため、当該チュートリアル視聴の際は注意が必要）
:::



ベグレリのグリップについては、スタンダードとフェイキーという概念も存在する。ビーズが手のひら側に来るグリップがスタンダード、ビーズが手の甲側に来るグリップがフェイキーである。

### プレーン
プレーンとは、ベグレリの回転面を表す概念である。主に以下の種類がある。
- バイシクルプレーン　（体に対してベグレリが垂直に回る。バイクに跨ったときの車輪の回転面と同じ）
- ウォールプレーン　（体に対してベグレリが平行に回る。壁を正面に立ったときの壁の面と同じ）
- フロアプレーン/0g（ゼロジー）　（ベグレリが床と平行に回る）

### 回転方向
ベグレリの回転方向は以下のように表現される
- インサイド・アップワード
- アウトサイド・ダウンワード
