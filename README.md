# Begleri Docs
ベグレリの様々な情報をまとめたドキュメントサイト

## 🖋️ ABOUT
現在のベグレリに関する情報や、大会・技術に関する情報を、時を経ても参照し続けることができるようにまとめるサイトです。サイトについて詳しくは[こちらのページ](/reference/about)を参照してください。
## 🚀 大会結果
ベグレリの2大大会であるSlingSlam・SingleGripOpenの歴代入賞者の入賞動画を確認することができます。過去の入賞者の傾向を見ながら自分の演技を考えることができます。
## ☀️ 姉妹サイト
姉妹サイトとして[BegleriInfo](https://sites.google.com/view/begleri-info/home)があります。BegleriInfoはベグレリを始める人向けに役立つサイトを志向している一方、本サイトは情報を記録することに焦点を当てています。
## 📄 サイトについて
高速な表示速度が特徴の[Astro](https://astro.build/)というフレームワークを使用しており、快適に閲覧できます。[Starlight](https://starlight.astro.build/)というドキュメントテーマを採用しているため、サイドバーや検索機能を使用して便利に閲覧することができます。
